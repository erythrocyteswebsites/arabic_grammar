المَبْنيُّ للْمَجْهُول

كَيْفَ نَبْنِي الْفِعْلَ الْمَاضِيَ لِلْمَجْهُولِ؟
  1. نَضُمُّ أَوَّلَهُ
  2. نَكْسِرُ مَا قَبْلَ آخِرِهِ
أَسْبَابِ حَذْفِ الْفَاعِلِ وَهِي
  1. الجهلُ بِهِ
  2. الْعِلْمُ بِهِ
  3. الْخَوْفُ عَلَيْهِ
  4. الْخَوْفُ مِنْهُ
  5. تَحْقِيرُ الْفَاعِل
  6. الإيجاز
  7. الْبَلَاغَةُ وَالسَّجْعُ
مَا التَّغَيُّرَاتُ الصَّرْفِيَّةُ الَّتِي تَحْدُثُ فِي الْفِعْلِ الَّذِي يُرَادُ بِنَاؤُهُ؟
الْمَاضِي

القاعدة: ضَمُّ الْأَوَّلِ وَكَسْرُ مَا قَبْلَ الْآخِرِ : [ كَتَبَ ← كُتِبَ, زَلْزَلَ ← زُلْزِلَ, أَكْرَمَ ← أُكرِمَ]

  1. السَّالِمُ وَالْمَهْمُوزُ وَالْمَثَالُ يَسِيرُ عَلَى الْقَاعِدَةِ بِلَا تَغْيِير
  2. إِذَا بَدَأَ الْفِعْلُ بِهمْرَةِ وَصْلٍ يُضَمُ ثَالِثَهُ مَعَ أَوْلِهِ مَعَ كَسْرِ ما قَبْلَ الآخِرِ: [ اِنْفَعَلَ ← اُنْفُعِلَ, اِفْتَعَلَ ← اُفْتُعِلَ, اِسْتَفْعَلَ ← اُسْتُفْعِلَ ]
  3. إِذَا بَدَأَ الْفِعْلُ بِتَاءٍ زَائِدَةٍ يُضَمُ ثَانِيهِ مَعَ أَوَّلِهِ ويُكْسَر مَا قَبْلَ آخِرِهِ: [ تَفَعَّلَ ← تُفُعِّلَ, تَفَاعَلَ ← تُفُوعِلَ, تَفَعلَلَ ← تُفُعْلِلَ ]
  4. الْفِعْلُ عَلَى وَزْنِ فَاعَلَ/تَفَاعَلَ تُقْلَبُ أَلِفُهُ وَاوًا بِسَبَبِ الضَّمّ: [ فَاعَلَ ← فُوعِلَ, تَفَاعَلَ ← تُفُوعِلَ ]
  5. الْمُضَعَّفُ الثَّلَاثِي: ضم الأول أو كسره: [ عَدَّ ← عِدَّ / عُدَّ ]
  6. مَزِيدُ الْمُضَعَّفَ:
    الرُّبَاعِيِّ وَالسُّدَاسِي يسير على القاعدة: [ أَعَدَّ ← أُعِدَّ ]
    الْخُمَاسِي أَوَّلِهِ فقط أو كَسْرُ مَا قَبْلَ آخِرِهِ فقط: [ اِعْتَدَّ ← اُعْتُدَّ / اِعْتِدَّ ]
  7. الْأَجْوَفُ الثَّلَاثِيُّ: [ قَالَ ← قِيلَ / قُولَ ]
  8. مَزِيدُ الْأَجْوَفِ:
    الرُّبَاعِيِّ ضَمُّ الْأَوَّلِ وَكَسْرُ مَا قَبْلَ الآخِرِ مَعَ قَلْبِ الْأَلِفِ يَاءً بِسَبَبٍ الكسْرةِ: [ أَعَادَ ← أُعِيدَ, اِسْتَعَانَ ← اُسْتُعِينَ ]
    الْخُمَاسِي [ اِحْتَارَ ← اُحْتُورَ / اِحْتِيرَ ]
  9. النَّاقِصُ (مُجَرَّدًا أَوْ مَزِيدًا): [ دَعا ← دُعِيَ, أَغْنَى ← أُغْنِيَ ]
الْمُضَارِعِ

القاعدة: ضَمُّ الْأَوَّلِ وَفَتْحُ مَا قَبْلَ الْآخِرِ:

  • يَفْعَلُ, يَفْعِلُ, يَفْعُلُ, يُفْعِل ← يُفْعَلُ,
  • يُفَعِّلُ ← يُفَعَّلُ,
  • يُفَاعِل ← يُفَاعَل,
  • يُفَعْلِلُ ← يُفَعْلَل,
  • يَنْفَعِلُ ← يُنْفَعَل
  • يَفْتَعِلُ ← يُفْتَعَلُ
  • يَفْعَلَّ ← يُفْعَلَّ
  • يَتَفَعَّلُ ← يُتَفَعَّلُ
  • يَتَفَاعَل ← يُتَفَاعَلُ
  • يَتَفَعْلَلُ ← يُتَفَعْلَلُ
  • يَسْتَفْعِلُ ← يُسْتَفْعَلُ
  • يَفْعَلِلَّ ← يُفْعَلَلَّ
  • يَفْعَنْلِلُ ← يُفْعَنْلَلُ
  1. السَّالِمُ وَالْمَهْمُوزِ والْمُضَعَّفَ (لَا يَحْدُثُ فِيهَا تَغْيِير): [ يَكْتُبُ ← يُكْتَبُ, يَشُدُّ ← يُشَدُّ, يُزَلْزِلُ ← يُزَلْزَلُ ]
  2. الْمِثَالَ ( تَعُودُ فَاءُ الْفِعْلِ الْمَحْذُوفَة): [ يَلِدُ ← يُولِدُ ]
  3. الْأَجْوَفُ وَالنَّاقِصُ مُجَرَّدًا أَوْ مَزِيدًا يُقْلَبُ حَرْفُ الْعِلَّةِ أَلِفَا بِسَبَبِ فَتْحِ مَا قَبْلَ الْآخِرِ:
    • يَقُولُ ← يُقَالُ,
    • يَبِيعُ ← يُبَاعُ,
    • يُرِيدُ ← يُرَادُ,
    • يَسْتَعِينُ ← يُسْتَعَانُ,
    • يَسْتَجِيبُ ← يُسْتَجَابُ,
    • يَمْشِي ← يُمْشَى,
    • يَدْعُو ← يُدْعَى,
    • يَفْتَرِي ← يُفْتَرَي,
    • يَسْتَغْنِي ← يُسْتَغْنَى
